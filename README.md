## What is GitLane?

GitLane is a powerful offline-first Git client for your Android device. Manage your repositories, commit changes, merge conflicts, and share code with peers - all without needing an internet connection!

---

## Key Features

✅ **Offline-First**: Work anywhere, anytime
✅ **Create Repositories**: Start new projects right from your phone
✅ **Commit Changes**: Track your work with meaningful commits
✅ **Merge Conflicts**: Handle conflicts with an easy-to-use interface
✅ **P2P Sharing**: Share code directly with nearby devices
✅ **GitHub Integration**: Sync with your GitHub repositories
✅ **Code Viewer**: Browse and edit files on the go

---

## Getting Started

1. **Download and Install**: Use the QR code or link above to install GitLane
2. **Set Up Your Identity**: Go to Settings and enter your name and email (required for commits)
3. **Create or Add a Repository**: Start a new repo or import an existing one
4. **Start Coding!**: Make changes, commit, and share with others
