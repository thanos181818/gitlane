# GitLane - Offline Git Client

GitLane is a powerful offline-first Git client for your Android device. It allows you to manage your repositories, commit changes, resolve merge conflicts, and share code with peers without requiring an internet connection.

## Key Features

* Offline-First: Work anywhere, anytime, without needing an internet connection.
* Create Repositories: Start new projects directly from your mobile device.
* Commit Changes: Track your work with standard Git commits.
* Merge Conflicts: Handle conflicts with a built-in interface.
* Peer-to-Peer Sharing: Share code directly with nearby devices over local networks.
* GitHub Integration: Sync with your GitHub repositories when an internet connection is available.
* Code Viewer: Browse and edit files on the go.

## Getting Started

1. Download and Install: Use the QR code or link above to install the GitLane application.
2. Set Up Your Identity: Navigate to Settings and enter your name and email. This is required for creating commits.
3. Create or Add a Repository: Start a new local repository or import an existing one.
4. Start Coding: Make changes, commit, and collaborate with others.
